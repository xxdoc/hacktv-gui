hacktv-gui is, as the name suggests, a GUI frontend application for hacktv. - https://github.com/fsphil/hacktv/ or https://github.com/captainjack64/hacktv/

This is the original version, written in Visual Basic 6.0. Works on virtually any version of Windows or on Wine with the VB6 runtimes installed. - https://www.microsoft.com/en-ie/download/details.aspx?id=24417
